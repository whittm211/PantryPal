PantryPal Support

Help for PantryPal pantry tracking, grocery lists, meal planning, and AI Chef suggestions.

Getting Help

For launch support, use the GitHub repository issue tracker or contact the PantryPal project owner.

When reporting a problem, include:

  • What screen you were on.
  • Whether you were using guest mode or a signed-in account.
  • What you expected to happen.
  • What actually happened.
  • Any visible error message.
  • Your browser and device, if known.

Privacy

Read the PantryPal privacy policy at privacy.html.

Data Export

Guest mode stores data on the current device. To move or back up guest data, open Settings > Data > Export JSON, then import that backup on another device.

Account And Sign-In Help

If email confirmation is enabled, confirm your email before signing in. If you requested too many confirmation emails, wait a few minutes before trying again.

Common Issues

I do not see my guest data on another device

Guest data is stored in the browser on the current device and does not sync until you use a signed-in account.

AI Chef keeps generating

AI Chef first tries the PantryPal backend. If the remote request times out, the app falls back to local pantry-aware suggestions so you can still get meal ideas.

Barcode lookup does not find my item

Barcode lookup depends on available product data. If an item is not found, enter the food details manually.

Known Limitations

  • Barcode coverage is incomplete.
  • Receipt scanning is not available yet.
  • Nutrition values are estimates when shown.
  • AI Chef suggestions should be reviewed for allergies and dietary needs.
  • Guest data can be lost if browser storage is cleared.

Security Notes

Do not share private Supabase keys, access tokens, or local environment files in support requests.
