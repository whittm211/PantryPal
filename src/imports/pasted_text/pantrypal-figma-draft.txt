Create a complete mobile app prototype draft in Figma for a project called PantryPal.

PantryPal is a mobile-first grocery tracking and simple meal planning app. The app helps users track food they already have, reduce food waste, create grocery lists, and find simple meal ideas based on pantry items.

Primary platform: Mobile app for iOS and Android
Secondary platform: Web app later

Create the Figma file as a polished class project draft with organized sections, clear page titles, and mobile app screens that can later be connected into a clickable prototype.

PROJECT GOAL

Build a realistic mobile prototype for PantryPal that shows the main user experience:
1. User enters the app.
2. User views the home dashboard.
3. User adds a food item.
4. User sees the item in the pantry.
5. User creates or updates a grocery list.
6. User views meal ideas based on available food.
7. User can understand expiring and low-stock food status.

OVERALL FIGMA STRUCTURE

Create these main Figma sections:

1. Project Overview
2. User Flow Summary
3. App Sitemap
4. Design System Draft
5. Mobile App Screens
6. Prototype Notes / Clickable Connections

Use clean spacing, organized frames, and readable labels. The mobile screens should be sized for iPhone 390 x 844.

APP STYLE

Use a clean, simple, beginner-friendly grocery app style.

Visual direction:
- Fresh and organized
- Soft rounded cards
- Large readable text
- Simple icons
- Clear buttons
- Mobile-first layout
- Friendly and practical, not overly fancy

Use the PantryPal design system:

Typography:
- Primary typeface: Inter
- Base body size: 16px
- Use clear hierarchy with large screen titles and medium card titles

Colors:
- Pantry Green: #3F7D58
- Fresh Mint: #A7D7A9
- Warm Cream: #FFF7E6
- Soft Sage: #DDEBDD
- Grocery Brown: #8A6A44
- Tomato Red: #D95D39 for expiration warnings
- Lemon Yellow: #F4C95D for low-stock alerts
- Sky Blue: #5DADEC for links or info
- Gray 900: #1F2933 for main text
- Gray 700: #3E4C59 for secondary text
- Gray 300: #CBD2D9 for borders
- Gray 100: #F5F7FA for light backgrounds
- White: #FFFFFF for cards and inputs

ACCESSIBILITY NOTES

Make the app readable and accessible:
- Body text should be at least 16px
- Buttons should be large and easy to tap
- Do not use color alone for alerts
- Expiring food should include warning labels/icons
- Low-stock items should include labels/icons
- Navigation should be simple and shallow
- Use clear labels, not vague icons only

SECTION 1: PROJECT OVERVIEW

Create a frame titled “PantryPal Project Overview.”

Include:
App Name: PantryPal

Short Description:
PantryPal is a mobile-first grocery tracking and simple meal planning app that helps users track food they already have, reduce food waste, create grocery lists, and find simple meal ideas based on pantry items.

Target Users:
- College students on tight budgets
- Single working adults who get home tired after work
- Families managing groceries for multiple people

Primary Platform:
Mobile app for iOS and Android

Secondary Platform:
Web app

Main User Need:
Users need a quick way to know what food they already have, what is close to expiring, what they need to buy, and what they can cook without overthinking meal planning.
